活动前言
铭记历史不是为了延续仇恨，而是为了更好地开创未来。

市民朋友们：

突如其来的新型冠状病毒肺炎,让2020年的春天格外不同。今年,我们用不一样的方式举行深圳市“中英街3·18警示日”活动，以网上鸣钟的形式警示后人铭记历史，开创未来。

1899年的今天，中英双方根据《中英展拓香港界址专条》完成了沙头角的勘界，中英街内的界碑见证了中华民族的百年屈辱。为了加强爱国主义教育，从2002年开始，我们在每年的3月18日举行鸣钟仪式，就是要警示人们铭记历史、勿忘国耻。19年来，我们不断创新形式、丰富内涵，活动影响不断扩大，取得了良好的社会效果，深圳市“中英街3·18警示日”已经成为深圳乃至全国有相当影响力的爱国主义教育品牌活动。

2020年是深圳经济特区建立40周年，也是建设粤港澳大湾区和深圳建设中国特色社会主义先行示范区全面铺开、纵深推进的关键之年。特别之年，特别之位，特别之为。当前正处于新型冠状病毒肺炎疫情防控的决胜阶段，深圳在抓好疫情防控的同时，统筹抓好改革发展稳定各项工作，以“先行示范”的担当作为，在高质量发展上迈出坚实步伐，率先从疫情影响中走出来，奋力夺取疫情防控和经济社会发展双胜利。

市民朋友们，站在新的历史起点上，我们要通过开展深圳市“中英街3·18警示日”等一系列爱国主义教育活动，弘扬中华民族的伟大民族精神，用历经磨难终未改、不屈不挠永向前的顽强品格，凝聚起众志成城、共克时艰的强大合力。我们要更加坚定中国特色社会主义的道路自信、理论自信、制度自信和文化自信，不断增强团结一心的精神纽带、自强不息的精神动力，继续朝着中华民族伟大复兴的中国梦奋勇前进！
